# Brand To DESIGN.md Checklist

## Evidence Checklist

- Official URL opened or fetched.
- Brand/press/newsroom/media-kit page checked when available.
- CSS font declarations checked.
- SVG logo colors checked when accessible.
- Key visual image downloaded or inspected when useful.
- Primary source limitations recorded.

## Token Checklist

- Color roles are named by purpose, not just hue.
- Fonts include fallback stacks.
- Components include at least buttons, cards, inputs, links.
- Layout rules include density, grid, and spacing behavior.
- Imagery rules describe motif, photography/illustration style, and what to avoid.
- Logo rules mention usage boundaries and permission limits when relevant.

## Demo Checklist

- First screen is a usable experience.
- Brand signal appears in first viewport.
- Demo includes at least three UI contexts.
- Desktop screenshot captured.
- Mobile screenshot captured.
- Console has no errors.
- `document.body.scrollWidth <= window.innerWidth`.
- Critical assets load, or fallback is acceptable.
